## Sway config for Macbook Air 7,2 (2017)
**Specs:**
- 8GB RAM
- Intel Quad Core i5-5350U
- Instel HD Graphics 6000
- 256 GB SSD

### Dependencies
**Arch**

sudo pacman -S swaybg swayidle swaylock mako

# yay needed to install the following

yay swayfx # Pick the first option
